# Pioneer Portfolio

Personal portfolio website built with Vue 3 + Vite.

This project showcases my work, profile, and contact details, and it is also structured so others can reuse it as a starter template for their own portfolio site.

## Tech Stack

- Vue 3
- Vite
- TypeScript
- Vue Router
- Tailwind CSS

## Requirements

- Node.js `^20.19.0 || >=22.12.0`
- npm, yarn, pnpm, or bun

## Getting Started

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

## Available Scripts

- `dev` - Run the Vite development server
- `build` - Type-check and build for production
- `preview` - Preview the built app
- `format` - Format files in `src/` with Prettier

## Use This as a Template

If you want to reuse this project for your own portfolio:

1. Replace content in `src/content/` with your own data.
2. Update images and assets in `src/assets/`.
3. Adjust page sections/components in `src/views/` and `src/components/`.
4. Update metadata in `index.html`.
